## 청년센터/프로그램 찾기 빠른 방법

1. 온통청년에서 지역 선택  
2. 키워드 검색  
   - `청년도전지원사업`
   - `구직단념청년`
   - `청년센터 프로그램`
   - `청년 서포터즈`
   - `청년 공모전`
   - `취업역량`
   - `마음건강/상담`
3. 각 지역 포털에서 모집 공고 원문 확인

---

## 공모전·서포터즈·대외활동 확인 루트

- 각 지자체 청년포털의 **공지/사업공고/대외활동** 메뉴
- 청년센터 공지사항(모집 일정, 대상, 신청링크)
- 공식 SNS(인스타/유튜브/블로그)는 보조 확인용으로 사용  
  → 최종 기준은 **홈페이지 원문 공고**

---

## 사용 시 주의사항

- 모집 일정/자격요건은 수시 변경될 수 있으니 반드시 원문 확인
- 신청 전 공식 도메인/주최기관 확인
- 개인정보 입력 시 피싱·사칭 링크 주의
- 본 문서는 **바로가기 링크 모음**이며, 세부 조건은 각 공고에 따릅니다.
